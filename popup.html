Etherscan Transaction Cancellation

This extension allows canceling pending transactions on Etherscan.

Cancellation is sending a transaction with zero value and the same nonce. An account should send the transaction to themself. Gas price for the transaction should be higher than for transaction which should be canceled.

The extension requires injected MetaMask as a wallet. Transactions are executed on behalf of the injected MetaMask account. The account controls gas price of the transaction by themself.

Warning: The cancelation will not work if the transaction was initiated from MetaMask. MetaMask ignores defined nonce, that is why it will not be possible. The transaction which was initiated from MetaMask can be canceled in MetaMask.
